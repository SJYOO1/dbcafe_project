# 따봉카페 프로젝트

## 프랜차이즈 카페의 매출을 조회하고 메뉴를 관리할 수 있는 시스템

### 목차
1. 팀원
2. 개발 순서
3. 사용 순서
4. 사용한 Oracle Tables

## 1. 팀원
- 김종민 : 프론트

- 윤건우 : 기간별 매출 조회

- 신재호 : 주문방식, 결제방식별 매출 및 판매량 조회

- 신혜수 : 카테고리별 매출 및 판매량 조회

- 유승주 : 로그인, 메뉴 관리 및 조회

***

## 2. 개발 순서
업무분석 -> 화면설계 -> 플로우차트 작성 -> 인터페이스 구성 -> DB구축 -> 구현 -> 오류 수정 -> 완성

***
